**1.3 添加用户**
--------------------

.. code-block:: vim

    点击“用户管理”下的“用户管理”菜单，进入用户管理页面。
    如果配置了ldap则在“新增”按钮后面会有一个“AD”按钮，如下图。点击“AD”按钮会从ldap获取用户
    如果没有配置ldap则需要手工为自己添加用户，是否管理员设置为“是”
    然后使用新的用户重新登录SSO

.. image:: ../_static/img/using/sso_user.png
